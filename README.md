# Ego — Software Developer

I am a 14-year-old software developer with a comprehensive understanding of
programming languages spanning systems-level, high-level, scripting, and
functional paradigms. I build across the full spectrum of software engineering
— from low-level systems code to full-stack web applications, security
research, artificial intelligence, and developer tooling.

My goal is simple: write software that matters, ship it in public, and
continuously raise the standard of what I produce.

---

## What I Do

**Systems & Low-Level Engineering**  
I work in Rust, C, and C++ to build performant, memory-safe software close to
the metal. I am interested in operating systems, compilers, and anything that
demands precision at the lowest level of the stack.

**Full-Stack Web Development**  
I build complete web applications from database to interface. I work across
modern frontend frameworks, backend runtimes, REST and WebSocket APIs, and
cloud infrastructure — whatever the product requires.

**Security & Exploit Development**  
I have a strong interest in offensive and defensive security, including
vulnerability research, reverse engineering, and building tools that expose
weaknesses before others do.

**Artificial Intelligence & Machine Learning**  
I apply machine learning to real problems — not as a buzzword, but as a
practical engineering tool. I work with model training, inference pipelines,
and integrating AI into production systems.

**Developer Tooling & Infrastructure**  
I build tools that make other developers faster. CLIs, automation scripts,
monitoring utilities, and open source libraries are a core part of what I
ship.

**Cryptocurrency & Simulation**  
I develop open source tooling for crypto simulation platforms, including
analytics dashboards, trading automation, and community utilities.
